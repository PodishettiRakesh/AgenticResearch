🟢 PHASE 3: Chunking Engine (2 hrs)
🎯 Goal:
Handle token limits

✅ Steps:
1. Chunk function
def chunk_text(text, size=1200, overlap=200):
   chunks = []
   for i in range(0, len(text), size - overlap):
       chunks.append(text[i:i+size])
   return chunks

2. Apply per section
chunks = {
 "methodology": [...],
 "results": [...],
 "figures": [...]
}

💡 Tip:
Don’t chunk abstract (keep whole)

🟢 PHASE 4: CrewAI Agent Setup (3–4 hrs)
🎯 Goal:
Create agents + tasks

✅ Define Agents

🧠 Consistency Agent
Role: Logical evaluator
Goal: Check method vs results

✍️ Grammar Agent
Role: Language reviewer

🧪 Novelty Agent
Role: Research evaluator

🔍 Fact-check Agent
Role: Claim verifier

✅ Define Tasks
Each task = one LLM call

Task 1: Consistency (Map-Reduce)
Loop through chunks → analyze
Final reduce → summary

Task 2: Grammar
Input: abstract

Task 3: Novelty
Input: abstract + conclusion

Task 4: Fact-check
Input: selected chunks

 👉 You can still manually control flow
